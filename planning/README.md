# Planning & Roadmap

This directory contains planning documents, roadmaps, and implementation guides for Browski development.

## Documents

### [roadmap.md](roadmap.md)
Long-term feature roadmap organized by release timeframes:
- **Current Status** (v1.0)
- **Short Term** (1-2 releases)
- **Medium Term** (3-6 months)
- **Long Term** (6-12 months)
- **Community & Documentation**
- **Technical Debt**

### [distribution.md](distribution.md)
Complete distribution implementation guide covering:
- **Implementation Status** - What's done vs. pending
- **Phase-by-Phase Plan** - Organized implementation steps
- **Platform-Specific Details** - macOS, Windows, Linux packaging
- **CI/CD Integration** - GitHub Actions workflows
- **Testing & Verification** - Acceptance criteria
- **Distribution Channels** - Homebrew, Chocolatey, Snap, etc.
